# Second Brain Status

Read automation state, source ledger, latest reports and proposals, recent operation log entries, and repository status. Report processed and pending sources, failed or review-required items, most recent successful ingestion and lint, unresolved errors and warnings, and any dirty or concurrent-work risk. Do not modify files.
