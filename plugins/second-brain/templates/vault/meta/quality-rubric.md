# Quality Rubric

Score each dimension from 0 to 2.

| Dimension | 0 | 1 | 2 |
|---|---|---|---|
| Provenance | Missing | Partial | Claim-level traceability |
| Canonicalization | Duplicate/ambiguous | Mostly consistent | Clear title and aliases |
| Epistemic clarity | Fact and inference mixed | Some labels | Evidence, inference, uncertainty separated |
| Connectivity | Orphaned | Some links | Useful inbound and outbound links |
| Freshness | Unknown/stale | Review needed | Current for available sources |
| Schema | Invalid | Minor drift | Fully conformant |

Pages scoring below 8/12 should be marked `needs-review`. A score never substitutes for independent verification.
