# Skill Structure

## Canonical Shape

```text
skills/
  <skill-name>/
    SKILL.md
    references/      # optional
    scripts/         # optional
    assets/          # optional
```

## Rules

- Skill file must be `SKILL.md` inside a dedicated skill directory.
- Keep supporting files colocated inside that skill directory.
- Use relative links from `SKILL.md`.
